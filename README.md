# 🌱 Explainable Smart Farming Decision System

## 📌 Overview

This project uses Machine Learning and Explainable AI to help farmers make better decisions regarding crop diseases and treatments.

## 🚀 Features

* Crop disease prediction using Deep Learning
* Explainability using Grad-CAM
* User-friendly interface (Streamlit/Flask)
* Real-time image input

## 🛠️ Tech Stack

* Python
* TensorFlow / Keras
* OpenCV
* Streamlit

## 📂 Project Structure

* app.py → Main application
* model/ → Trained model files
* utils/ → Helper functions

## ▶️ How to Run

```bash
pip install -r requirements.txt
python app.py
```
